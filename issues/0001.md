|---|---|

## Description

The package only depends on swift-argument-parser, has no platform constraint, and produces an executable named `NeoPager`. SwiftTUI must be added and the package configured before any TUI code can be written.

## Expected behavior

`Package.swift` declares `platforms: [.macOS(.v13)]`, adds the SwiftTUI dependency (`https://github.com/rensbreur/SwiftTUI.git`, pinned to branch `main`), threads the product into the target, applies `swiftSettings: [.defaultIsolation(MainActor.self)]`, and declares an executable product named `neopager` (lowercase, the installed command name).

## Notes

Mirror the configuration in `../TerminalDashboard/Package.swift`, which uses exactly this setup (SwiftTUI on branch `main`, macOS 13+, MainActor default isolation, Swift 6 language mode). SwiftTUI has historically not tagged semver releases, hence the branch pin. Verify with `./build.sh clean build`.
